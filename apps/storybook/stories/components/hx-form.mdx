## Overview

The `hx-form` component is a wrapper around the native `<form>` element that provides standardized layout,
spacing, and event handling for HELIX form controls. It automatically manages form submission, integrates
with ElementInternals-based form controls, and emits custom events for validation and submit actions.

Unlike a basic `<form>`, `hx-form` provides:

- Consistent vertical spacing between form fields
- Automatic coordination of form-associated custom elements
- Custom `hx-submit` events with structured data payloads
- Built-in validation state management
- Responsive layout patterns for multi-column forms

### When to Use

- **Patient intake forms** — Collect patient demographics, insurance, and medical history
- **Order entry** — Submit medication orders, lab requisitions, or imaging requests
- **Referral workflows** — Capture referral details, priority, and department
- **Admin configuration** — Settings forms, user profile updates, or system preferences
- **Search/filter UIs** — Multi-field search forms for patient lookup or report generation

### When Not to Use

- **Single input forms** — Use a standalone input with a button for simple search or filter
- **Non-submitting UIs** — If the form never submits data, use semantic HTML (`<div>`, `<section>`)
- **Read-only data display** — Use cards or data grids, not forms

## Accessibility

The Form component follows WCAG 2.1 AA guidelines:

- **Keyboard navigation** — Full support for Tab, Shift+Tab, Enter (submit)
- **Screen reader support** — Proper `<form>` semantics, fieldset/legend for grouping
- **Validation announcements** — Error messages are announced to assistive technologies
- **Focus management** — Focus moves to the first invalid field on validation failure

## Properties

| Property     | Type                                   | Default      | Description                                               |
| ------------ | -------------------------------------- | ------------ | --------------------------------------------------------- |
| `layout`     | `'vertical' \| 'horizontal' \| 'grid'` | `'vertical'` | Layout mode for form fields                               |
| `gap`        | `'sm' \| 'md' \| 'lg'`                 | `'md'`       | Spacing between form fields                               |
| `name`       | `string`                               | `''`         | Name attribute for the form                               |
| `action`     | `string`                               | `''`         | Form action URL (rarely used with custom submit handlers) |
| `method`     | `'get' \| 'post'`                      | `'post'`     | HTTP method for form submission                           |
| `novalidate` | `boolean`                              | `false`      | Disable native HTML5 validation                           |

## Slots

| Slot        | Description                                 |
| ----------- | ------------------------------------------- |
| `(default)` | Form fields and content                     |
| `actions`   | Form action buttons (Submit, Cancel, Reset) |

## CSS Custom Properties

Override these tokens to theme the form:

| Property              | Default             | Description                            |
| --------------------- | ------------------- | -------------------------------------- |
| `--hx-form-gap`       | `var(--hx-space-4)` | Gap between form fields                |
| `--hx-form-max-width` | `100%`              | Maximum width of the form container    |
| `--hx-form-padding`   | `0`                 | Internal padding of the form container |

## CSS Parts

Style internal form elements using `::part()`:

| Part      | Description                      |
| --------- | -------------------------------- |
| `form`    | The native `<form>` element      |
| `fields`  | The container for form fields    |
| `actions` | The container for action buttons |

## Events

| Event        | Detail                                                | Description                         |
| ------------ | ----------------------------------------------------- | ----------------------------------- |
| `hx-submit`  | `{ formData: FormData, values: Record<string, any> }` | Fired when the form is submitted    |
| `hx-reset`   | `{}`                                                  | Fired when the form is reset        |
| `hx-invalid` | `{ field: HTMLElement, message: string }`             | Fired when a field fails validation |

## Form Participation

The `hx-form` component integrates with all HELIX form controls that use ElementInternals:

- `hx-text-input`
- `hx-textarea`
- `hx-select`
- `hx-checkbox`
- `hx-switch`
- `hx-radio-group`

These components automatically participate in form submission and validation when placed inside `hx-form`.

## Design Guidelines

### Form Layout Selection

- **Vertical (default)** — Best for most forms. Each field gets full width, easy to scan.
- **Horizontal** — Search bars and single-field forms where label and input sit side-by-side.
- **Grid** — Multi-column forms for dense data entry (e.g., patient demographics).

### Spacing

- Use `gap="sm"` for compact forms (filter panels, inline search)
- Use `gap="md"` (default) for standard forms
- Use `gap="lg"` for forms with complex fields or long labels

### Button Placement

- Place primary action (Submit) on the left, secondary actions (Cancel, Reset) on the right
- Always use `type="submit"` for the primary button to enable Enter key submission
- Avoid more than 3 action buttons per form

### Validation Timing

- Validate on submit, not on blur (to avoid interrupting user flow)
- Show inline errors after the first submit attempt
- Clear errors when the user corrects the field

### Drupal Behaviors for Form Handling

```javascript
/**
 * @file
 * hx-form Drupal behavior.
 */

(function (Drupal, once, drupalSettings) {
  'use strict';

  /**
   * Attach hx-form event listeners.
   */
  Drupal.behaviors.hxForm = {
    attach: function (context, settings) {
      // Initialize all HELIX forms
      once('hx-form-init', 'hx-form', context).forEach((form) => {
        const formId = form.getAttribute('id');

        // Listen for form submission
        form.addEventListener('hx-submit', async (e) => {
          e.preventDefault();

          const { formData, values } = e.detail;

          console.log('Form submitted:', formId, values);

          // Option 1: Standard POST submission
          submitFormStandard(form, formData);

          // Option 2: AJAX submission
          // await submitFormAjax(form, formData);

          // Option 3: Drupal AJAX API
          // submitFormDrupalAjax(form, values);
        });

        // Listen for form reset
        form.addEventListener('hx-reset', (e) => {
          console.log('Form reset:', formId);

          // Clear any custom errors
          form.querySelectorAll('[error]').forEach((field) => {
            field.removeAttribute('error');
          });
        });

        // Listen for validation failures
        form.addEventListener('hx-invalid', (e) => {
          const { field, message } = e.detail;

          console.error('Validation failed:', field.getAttribute('name'), message);

          // Display error on the field
          field.setAttribute('error', message);

          // Scroll to first error
          field.scrollIntoView({ behavior: 'smooth', block: 'center' });
        });
      });
    },

    detach: function (context, settings, trigger) {
      if (trigger === 'unload') {
        context.querySelectorAll('hx-form').forEach((form) => {
          form.removeEventListener('hx-submit', () => {});
          form.removeEventListener('hx-reset', () => {});
          form.removeEventListener('hx-invalid', () => {});
        });
      }
    },
  };

  /**
   * Submit form using standard POST.
   */
  function submitFormStandard(form, formData) {
    const action = form.getAttribute('action') || window.location.href;
    const method = form.getAttribute('method') || 'POST';

    fetch(action, {
      method: method,
      body: formData,
      headers: {
        'X-Requested-With': 'XMLHttpRequest',
      },
    })
      .then((response) => response.json())
      .then((data) => {
        if (data.success) {
          window.location.href = data.redirect || '/';
        } else {
          displayFormErrors(form, data.errors);
        }
      })
      .catch((error) => {
        console.error('Form submission error:', error);
        alert('An error occurred. Please try again.');
      });
  }

  /**
   * Submit form using Drupal AJAX API.
   */
  function submitFormDrupalAjax(form, values) {
    const ajaxSettings = {
      url: form.getAttribute('action'),
      submit: values,
    };

    const ajax = Drupal.ajax(ajaxSettings);
    ajax.execute();
  }

  /**
   * Display server-side validation errors.
   */
  function displayFormErrors(form, errors) {
    Object.keys(errors).forEach((fieldName) => {
      const field = form.querySelector(`[name="${fieldName}"]`);
      if (field) {
        field.setAttribute('error', errors[fieldName]);
      }
    });
  }
})(Drupal, once, drupalSettings);
```

### Multi-Step Form Workflow

**Wizard-style form with step progression:**

```javascript
(function (Drupal, once) {
  Drupal.behaviors.hxFormWizard = {
    attach: function (context) {
      once('hx-form-wizard', '[data-wizard]', context).forEach((wizard) => {
        const steps = wizard.querySelectorAll('[data-step]');
        let currentStep = 0;

        // Initialize wizard
        showStep(currentStep);

        // Listen for form submissions in each step
        steps.forEach((step, index) => {
          const form = step.querySelector('hx-form');

          if (!form) return;

          form.addEventListener('hx-submit', (e) => {
            e.preventDefault();

            // Validate current step
            if (validateStep(form)) {
              // Save step data to sessionStorage
              saveStepData(index, e.detail.values);

              // Move to next step or submit
              if (index < steps.length - 1) {
                currentStep++;
                showStep(currentStep);
              } else {
                submitWizard(wizard);
              }
            }
          });
        });

        function showStep(stepIndex) {
          steps.forEach((step, index) => {
            step.style.display = index === stepIndex ? 'block' : 'none';
          });

          // Update progress indicator
          updateProgress(stepIndex, steps.length);
        }

        function validateStep(form) {
          // Check native HTML5 validation
          const inputs = form.querySelectorAll('[required]');
          let isValid = true;

          inputs.forEach((input) => {
            if (!input.value) {
              input.setAttribute('error', 'This field is required.');
              isValid = false;
            }
          });

          return isValid;
        }

        function saveStepData(stepIndex, data) {
          const wizardData = JSON.parse(sessionStorage.getItem('wizardData') || '{}');
          wizardData[`step_${stepIndex}`] = data;
          sessionStorage.setItem('wizardData', JSON.stringify(wizardData));
        }

        function submitWizard(wizard) {
          const allData = JSON.parse(sessionStorage.getItem('wizardData') || '{}');

          // Flatten all step data
          const combinedData = Object.values(allData).reduce((acc, stepData) => {
            return { ...acc, ...stepData };
          }, {});

          // Submit combined data
          fetch('/api/wizard-submit', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(combinedData),
          })
            .then((response) => response.json())
            .then((data) => {
              sessionStorage.removeItem('wizardData');
              window.location.href = data.redirect;
            })
            .catch((error) => {
              console.error('Wizard submission error:', error);
            });
        }

        function updateProgress(current, total) {
          const progress = ((current + 1) / total) * 100;
          const progressBar = wizard.querySelector('[data-progress-bar]');
          if (progressBar) {
            progressBar.style.width = `${progress}%`;
          }
        }
      });
    },
  };
})(Drupal, once);
```
