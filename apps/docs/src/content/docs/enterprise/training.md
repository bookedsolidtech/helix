HELIX offers structured training materials and a certification program for developers, technical leads, and organizations building on the library. Training is available as self-paced online content, instructor-led workshops, and on-site engagements for enterprise customers.

## Certification Paths

### HELIX Developer Certification

For individual contributors building components or integrating HELIX into existing systems.

**Prerequisites:** Working knowledge of HTML, CSS, JavaScript, and basic TypeScript.

| Module | Topics | Format | Duration |
| --- | --- | --- | --- |
| 01 — Foundations | Web Components basics, Shadow DOM, Custom Elements registry | Self-paced | 3 hours |
| 02 — HELIX Core | Installing HELIX, using components, design tokens, theming | Self-paced | 4 hours |
| 03 — Component Development | Lit 3.x, reactive properties, lifecycle, template syntax | Self-paced | 6 hours |
| 04 — Accessibility | WCAG 2.1 AA, ARIA patterns, keyboard navigation, testing | Self-paced | 4 hours |
| 05 — Testing | Vitest browser mode, shadow DOM testing, accessibility audits | Self-paced | 3 hours |
| 06 — Capstone | Build and publish a compliant HELIX component | Graded project | 4–6 hours |

**Certification exam:** 60 questions, 90 minutes, 75% passing score. Valid for 2 years.

---

### HELIX Drupal Integration Certification

For Drupal developers integrating HELIX into Twig-based sites.

**Prerequisites:** HELIX Developer Certification or equivalent experience, Drupal 10+ familiarity.

| Module | Topics | Format | Duration |
| --- | --- | --- | --- |
| 01 — Drupal Fundamentals | Library system, theme layer, Twig templating | Self-paced | 3 hours |
| 02 — HELIX in Twig | Properties, slots, attributes, event handling in Twig | Self-paced | 4 hours |
| 03 — Drupal Behaviors | Drupal.behaviors lifecycle, Once API, web component initialization | Self-paced | 3 hours |
| 04 — SDC and Form API | Single Directory Components, Form API element plugins | Self-paced | 3 hours |
| 05 — Performance | Per-component loading, lazy loading, CDN vs npm | Self-paced | 2 hours |
| 06 — Capstone | Integrate HELIX components into a Drupal theme | Graded project | 4–6 hours |

**Certification exam:** 50 questions, 75 minutes, 75% passing score. Valid for 2 years.

### HELIX Architect Certification

For technical leads and architects designing HELIX-based design systems at the organizational level.

**Prerequisites:** HELIX Developer Certification and 6+ months of production HELIX experience.

| Module | Topics | Format | Duration |
| --- | --- | --- | --- |
| 01 — Design System Architecture | Token system design, multi-brand strategy, governance | Instructor-led | 4 hours |
| 02 — Enterprise Theming | Theme packages, token override strategy, dark mode, brand kits | Instructor-led | 4 hours |
| 03 — Monorepo Patterns | Turborepo, workspace structure, shared tooling, CI/CD | Instructor-led | 3 hours |
| 04 — Accessibility Program | Organizational a11y strategy, audit cadence, remediation workflow | Instructor-led | 3 hours |
| 05 — Healthcare Compliance | WCAG in clinical contexts, HIPAA considerations, audit documentation | Instructor-led | 4 hours |
| 06 — Capstone | Architecture review of a real design system proposal | Graded presentation | 2–4 hours |

**Certification exam:** Live oral defense with HELIX core team reviewer. Valid for 2 years.

- **Code examples repository** — Annotated example projects on GitHub

## Organizational Training Programs

For organizations onboarding multiple teams, HELIX offers:

- **Cohort-based learning** — All developers complete the certification path together with a shared cohort Slack channel
- **Custom curriculum** — Modules tailored to your tech stack and compliance requirements
- **Train-the-trainer** — Certify internal instructors to deliver HELIX training independently
- **Progress dashboard** — Track certification status across your organization

Contact [training@wc-2026.dev](mailto:training@wc-2026.dev) to design a program for your organization.

## Recertification

Certifications are valid for 2 years. Recertification options:

- Complete a 2-hour recertification module covering changes since your last certification
- Attend any HELIX conference session and complete the post-session assessment
- Pass the full certification exam again

Certified developers are listed in the HELIX certified professional directory (opt-in).
