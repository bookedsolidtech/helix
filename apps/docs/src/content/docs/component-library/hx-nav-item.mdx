## Overview

`hx-nav-item` is a navigation item designed for use inside `hx-side-nav`. It renders as an anchor (`<a>`) when a URL is provided, or a button (`<button>`) when it has nested child navigation. It supports icons, badges, active/disabled states, sub-navigation expand/collapse, and collapsed-mode tooltips with `aria-describedby`.

**Use `hx-nav-item` when:** building navigation inside `hx-side-nav` — clinical portals, admin dashboards, department navigation trees.

For full usage context including keyboard navigation and collapsed mode behavior, see the [hx-side-nav documentation](/component-library/hx-side-nav/).

## Properties

| Property   | Attribute  | Type      | Default | Description                                                                                            |
| ---------- | ---------- | --------- | ------- | ------------------------------------------------------------------------------------------------------ |
| `href`     | `href`     | `string`  | `''`    | The URL this item links to. When set (and no children), renders as `<a>`.                              |
| `active`   | `active`   | `boolean` | `false` | Whether this item is the current/active page. Sets `aria-current="page"`. Reflects to attribute.       |
| `expanded` | `expanded` | `boolean` | `false` | Whether the sub-navigation panel is expanded. Reflects to attribute.                                   |
| `disabled` | `disabled` | `boolean` | `false` | Whether this item is disabled. Sets `aria-disabled="true"` and `tabindex="-1"`. Reflects to attribute. |

## Slots

| Slot        | Description                                                                        |
| ----------- | ---------------------------------------------------------------------------------- |
| _(default)_ | The label text for this navigation item.                                           |
| `icon`      | An icon displayed before the label. Should be an SVG or icon component.            |
| `badge`     | Badge content (e.g. notification count). Hidden automatically in collapsed mode.   |
| `children`  | Nested `hx-nav-item` elements for sub-navigation. Presence triggers button render. |

## CSS Parts

| Part       | Description                                               |
| ---------- | --------------------------------------------------------- |
| `link`     | The `<a>` or `<button>` element (the interactive part).   |
| `icon`     | The icon container (wraps the `icon` slot).               |
| `label`    | The label container (wraps the default slot text).        |
| `badge`    | The badge container (wraps the `badge` slot).             |
| `children` | The sub-navigation container (wraps the `children` slot). |

## Accessibility

| Topic              | Details                                                                                                                             |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| Element type       | Renders as `<a>` when `href` is set without children; `<button>` when children are present or no `href`.                            |
| `aria-current`     | Active anchors set `aria-current="page"` to indicate the current page to screen readers.                                            |
| `aria-disabled`    | Disabled items set `aria-disabled="true"`. Anchors use `tabindex="-1"` since native `disabled` is invalid on `<a>`.                 |
| `aria-expanded`    | Button items with children reflect `aria-expanded` for sub-navigation state.                                                        |
| `role="group"`     | The children container uses `role="group"` to semantically group nested items.                                                      |
| Collapsed tooltips | In collapsed mode, a tooltip (`role="tooltip"`) is rendered and referenced via `aria-describedby` for screen reader identification. |
| WCAG 2.1 AA        | Verified with axe-core: zero violations in default and active states.                                                               |
