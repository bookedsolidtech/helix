`hx-tab-panel` is a sub-component of [`hx-tabs`](/component-library/hx-tabs). Full usage documentation, live demos, accessibility details, and Drupal integration examples are on the **[hx-tabs documentation page](/component-library/hx-tabs)**.

## Quick Reference

`<hx-tab-panel>` must be a direct child of `<hx-tabs>` in the default slot. The `name` attribute must match the `panel` attribute on the corresponding `<hx-tab>`.

```html
<hx-tabs label="Example tabs">
  <hx-tab slot="tab" panel="one">Tab One</hx-tab>
  <hx-tab slot="tab" panel="two">Tab Two</hx-tab>
  <hx-tab-panel name="one"><p>Panel one content.</p></hx-tab-panel>
  <hx-tab-panel name="two"><p>Panel two content.</p></hx-tab-panel>
</hx-tabs>
```

Active panels have `tabindex="0"` and are focusable. Inactive panels have `hidden` and `tabindex="-1"`, removing them from the accessibility tree. Both attributes are managed by the parent `<hx-tabs>`.
