All available slots: image, heading, body, footer, and actions.

    <span slot="footer">Published Feb 2026</span>
    <div slot="actions">
      <wc-button variant="primary" size="sm">Action</wc-button>
      <wc-button variant="ghost" size="sm">Cancel</wc-button>
    </div>
  </wc-card>
</ComponentDemo>

## Interactive Card

Card with `wc-href` becomes clickable. Uses the `wc-` prefix to avoid conflicting with the native HTML `href` attribute.

<ComponentDemo title="Interactive Card">
  <wc-card variant="default" elevation="raised" wc-href="https://wc-2026.dev">
    <span slot="heading">Clickable Card</span>
    <p>This card has a <code>wc-href</code> attribute — click anywhere to navigate.</p>
  </wc-card>
</ComponentDemo>
