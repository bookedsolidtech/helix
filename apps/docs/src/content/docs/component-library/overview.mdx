The HELIX component library ships **production-ready Lit 3.x web components** with design tokens, form association, and full accessibility. Every component on this page is **rendered live** in your browser.

## Installation

```bash
npm install @helix/library
```

```html
<!-- Use anywhere — no framework required -->
<script type="module">
  import '@helix/library';
</script>

<hx-button variant="primary">Get Started</hx-button>
```

## Available Components

| Component                                         | Tag                | Description                                                                |
| ------------------------------------------------- | ------------------ | -------------------------------------------------------------------------- |
| [Alert](/component-library/hx-alert/)             | `<hx-alert>`       | Feedback messages for status, warnings, and errors with ARIA semantics     |

| [Form](/component-library/hx-form/)               | `<hx-form>`        | Form wrapper with validation, submission handling, and field coordination  |
| [Prose](/component-library/hx-prose/)             | `<hx-prose>`       | Rich text container with typographic styling for long-form content         |
| [Radio](/component-library/hx-radio/)             | `<hx-radio>`       | Single radio input with label and form association                         |
| [Radio Group](/component-library/hx-radio-group/) | `<hx-radio-group>` | Grouped radio inputs with fieldset semantics and shared validation         |
| [Select](/component-library/hx-select/)           | `<hx-select>`      | Native select wrapper with label, validation, and help text                |
| [Switch](/component-library/hx-switch/)           | `<hx-switch>`      | Toggle switch for binary on/off settings with label support                |

| [Textarea](/component-library/hx-textarea/)       | `<hx-textarea>`    | Multi-line text input with label, validation, and character count          |

## Key Features

- **Framework-agnostic** — works with React, Vue, Angular, Svelte, or vanilla HTML
- **Design token-driven** — themed via CSS custom properties
- **Form-associated** — form components participate in native `<form>` elements via ElementInternals
- **Accessible** — ARIA attributes, keyboard navigation, and focus management built in
- **Lightweight** — Lit 3.x with zero external dependencies beyond the framework
- **87 components** — from layout primitives to full form controls
