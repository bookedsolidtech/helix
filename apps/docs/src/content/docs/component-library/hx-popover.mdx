## Overview

`hx-popover` is a floating content panel anchored to a trigger element. It supports multiple trigger modes (click, hover, focus, manual), 12 placement options via Floating UI, an optional arrow indicator, and full accessibility with focus management and Escape key dismissal.

**Use `hx-popover` when:** you need to display rich, interactive content attached to a trigger element (patient details, contextual help, form previews).
**Use `hx-tooltip` instead when:** you need a simple text label on hover/focus with no interactive content.

## Basic Usage

```html
<hx-popover placement="bottom" trigger="click" label="Help text">
  <button slot="anchor">Open Popover</button>
  <p>Rich popover content here.</p>
</hx-popover>
```

## Properties

| Property    | Attribute   | Type               | Default     | Description                                             |
| ----------- | ----------- | ------------------ | ----------- | ------------------------------------------------------- |
| `open`      | `open`      | `boolean`          | `false`     | Whether the popover is open.                            |
| `placement` | `placement` | `PopoverPlacement` | `'bottom'`  | Preferred placement relative to the anchor (12 values). |
| `trigger`   | `trigger`   | `TriggerMode`      | `'click'`   | How the popover is triggered.                           |
| `distance`  | `distance`  | `number`           | `8`         | Distance in pixels between popover and anchor.          |
| `skidding`  | `skidding`  | `number`           | `0`         | Alignment offset in pixels along the anchor.            |
| `arrow`     | `arrow`     | `boolean`          | `false`     | Whether to show an arrow pointing to the anchor.        |
| `label`     | `label`     | `string`           | `'Popover'` | Accessible label for the popover body (`aria-label`).   |

### Placement Values

`top` | `top-start` | `top-end` | `right` | `right-start` | `right-end` | `bottom` | `bottom-start` | `bottom-end` | `left` | `left-start` | `left-end`

### Trigger Modes

| Mode     | Description                                        |
| -------- | -------------------------------------------------- |
| `click`  | Toggle on click. Click outside or Escape to close. |
| `hover`  | Show on mouseenter, hide on mouseleave.            |
| `focus`  | Show on focusin, hide on focusout.                 |
| `manual` | Controlled externally via the `open` property.     |

## Events

| Event           | Description                                    |
| --------------- | ---------------------------------------------- |
| `hx-show`       | Dispatched when the popover begins to show.    |
| `hx-after-show` | Dispatched after the popover is fully visible. |
| `hx-hide`       | Dispatched when the popover begins to hide.    |
| `hx-after-hide` | Dispatched after the popover is fully hidden.  |

## Slots

| Slot        | Description                                 |
| ----------- | ------------------------------------------- |
| `anchor`    | The trigger element that opens the popover. |
| _(default)_ | Content displayed inside the popover body.  |

## Accessibility

| Topic            | Details                                                                          |
| ---------------- | -------------------------------------------------------------------------------- |
| ARIA role        | `role="region"` on the popover body with `aria-label`.                           |
| `aria-expanded`  | Set on the anchor element, tracks open/close state.                              |
| `aria-hidden`    | Body has `aria-hidden="true"` when closed, `"false"` when open.                  |
| `inert`          | Body is marked `inert` when hidden, preventing tab focus into invisible content. |
| Focus management | Focus moves into the popover body on open, returns to trigger on close.          |
| Escape key       | Pressing Escape anywhere on the document closes the popover.                     |
| Click outside    | Clicking outside the component closes the popover (click trigger mode).          |
| Reduced motion   | Transitions disabled when `prefers-reduced-motion: reduce` is active.            |
| WCAG 2.1 AA      | Verified with axe-core in both closed and open states.                           |

## Drupal Integration

```twig
{# my-module/templates/my-template.html.twig #}
<hx-popover
  placement="{{ popover_placement|default('bottom') }}"
  trigger="{{ popover_trigger|default('click') }}"
  label="{{ popover_label }}"
  {% if popover_arrow %}arrow{% endif %}
>
  <button slot="anchor">{{ trigger_text }}</button>
  {{ popover_content }}
</hx-popover>
```

Load the library in your module's `.libraries.yml`:

```yaml
helix-components:
  js:
    /libraries/helix/helix.min.js: { minified: true }
```

Listen for popover events in Drupal behaviors:

```javascript
Drupal.behaviors.myPopover = {
  attach(context) {
    once('myPopover', 'hx-popover', context).forEach((el) => {
      el.addEventListener('hx-show', () => {
        // popover opened
      });
      el.addEventListener('hx-hide', () => {
        // popover closed
      });
    });
  },
};
```
