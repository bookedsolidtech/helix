## Overview

`hx-radio` is a presentational radio button designed to be used exclusively inside an `<hx-radio-group>`. The parent group manages selection, keyboard navigation, roving tabindex, and form value. Individual radios dispatch an internal `hx-radio-select` event that the group intercepts.

**Always use inside `<hx-radio-group>`** — standalone `hx-radio` elements lack keyboard navigation and form participation.

## Live Demo

### Inside a Radio Group

Radio buttons are designed to be used within an `<hx-radio-group>` for proper keyboard navigation and form association.

<ComponentDemo title="Inside a Radio Group">
  <hx-radio-group label="Select a department" name="dept-demo" value="eng">
    <hx-radio value="eng" label="Engineering"></hx-radio>
    <hx-radio value="design" label="Design"></hx-radio>
    <hx-radio value="product" label="Product"></hx-radio>
  </hx-radio-group>
</ComponentDemo>

## Properties

| Property   | Attribute  | Type      | Default | Description                                         |
| ---------- | ---------- | --------- | ------- | --------------------------------------------------- |
| `value`    | `value`    | `string`  | `''`    | The value this radio represents.                    |
| `label`    | `label`    | `string`  | `''`    | Visible label text. Override with the default slot. |
| `disabled` | `disabled` | `boolean` | `false` | Whether this radio is disabled.                     |
| `checked`  | `checked`  | `boolean` | `false` | Whether this radio is checked. Managed by parent.   |

## CSS Custom Properties

| Property                          | Default                                | Description               |
| --------------------------------- | -------------------------------------- | ------------------------- |
| `--hx-radio-size`                 | `var(--hx-size-5, 1.25rem)`            | Radio circle size.        |
| `--hx-radio-border-color`         | `var(--hx-color-neutral-300, #ced4da)` | Radio border color.       |
| `--hx-radio-checked-bg`           | `var(--hx-color-primary-500, #2563EB)` | Checked background color. |
| `--hx-radio-checked-border-color` | `var(--hx-color-primary-500, #2563EB)` | Checked border color.     |
| `--hx-radio-dot-color`            | `var(--hx-color-neutral-0, #ffffff)`   | Inner dot color.          |
| `--hx-radio-focus-ring-color`     | `var(--hx-focus-ring-color, #2563EB)`  | Focus ring color.         |
| `--hx-radio-label-color`          | `var(--hx-color-neutral-700, #343a40)` | Label text color.         |

## CSS Parts

| Part    | Description              |
| ------- | ------------------------ |
| `radio` | The visual radio circle. |
| `label` | The label text area.     |

## Slots

| Slot        | Description                                           |
| ----------- | ----------------------------------------------------- |
| _(default)_ | Custom label content. Overrides the `label` property. |

## Accessibility

| Topic           | Details                                                                             |
| --------------- | ----------------------------------------------------------------------------------- |
| ARIA role       | `role="radio"` set on the host element.                                             |
| `aria-checked`  | Synced with the `checked` property (`"true"` or `"false"`).                         |
| `aria-disabled` | Synced with the `disabled` property.                                                |
| Focus           | Managed by parent group via roving tabindex. Focus ring on `:host(:focus-visible)`. |
| Keyboard        | All keyboard interaction handled by the parent `<hx-radio-group>`.                  |
| WCAG 2.1 AA     | Verified with axe-core when used inside a radio group.                              |
