Different resize behaviors: none, vertical, both, and auto-grow.

  <div style="display: grid; gap: 1rem; max-width: 480px;">

  </div>

## Character Count

Show remaining characters with `maxlength` and `show-count`.

<ComponentDemo title="Character Count">
  <div style="max-width: 480px;">
    <hx-textarea
      label="Short bio"
      placeholder="Tell us about yourself..."
      maxlength="200"
      show-count
      rows="3"
      help-text="Keep it brief."
    ></hx-textarea>
  </div>
</ComponentDemo>
