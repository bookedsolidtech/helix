## Full Slot Usage

All available slots: image, heading, body, footer, and actions.

<ComponentDemo title="Full Slot Usage">
  <hx-card variant="default" elevation="raised">
    <div

    >
      Image Slot
    </div>
    <span slot="heading">Card Title</span>
    <p>This is the default body slot with descriptive content for the card.</p>
    <span slot="footer">Published Feb 2026</span>
    <div slot="actions">
      <hx-button variant="primary" size="sm">
        Action
      </hx-button>
      <hx-button variant="ghost" size="sm">
        Cancel
      </hx-button>
    </div>
  </hx-card>
</ComponentDemo>

## Interactive Card

Card with `hx-href` becomes clickable. Uses the `hx-` prefix to avoid conflicting with the native HTML `href` attribute.

<ComponentDemo title="Interactive Card">
  <hx-card variant="default" elevation="raised" hx-href="https://helix.dev">
    <span slot="heading">Clickable Card</span>
    <p>
      This card has a <code>hx-href</code> attribute — click anywhere to navigate.
    </p>
  </hx-card>
</ComponentDemo>
