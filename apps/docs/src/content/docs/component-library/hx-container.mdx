---
title: 'hx-container'
description: 'A layout container that constrains content width and provides consistent vertical spacing and horizontal gutters. Uses a two-layer model: the outer host spans full width (background, vertical padding), while the inner wrapper constrains max-width, centers content, and applies horizontal gutters.'
