## Variants

Info, success, warning, and error styles with appropriate ARIA semantics.

<ComponentDemo title="Variants">
  <div style="display: grid; gap: 0.75rem;">
    <hx-alert variant="info">This is an informational message.</hx-alert>
    <hx-alert variant="success">Operation completed successfully.</hx-alert>
    <hx-alert variant="warning">Please review before proceeding.</hx-alert>
    <hx-alert variant="error">An error occurred. Please try again.</hx-alert>
  </div>
</ComponentDemo>

## Dismissible

Alerts with a dismiss button. Click the close icon to hide the alert.

<ComponentDemo title="Dismissible">
  <div style="display: grid; gap: 0.75rem;">
    <hx-alert variant="info" dismissible>
      This alert can be dismissed by the user.
    </hx-alert>
    <hx-alert variant="success" dismissible>
      Success! You can close this notification.
    </hx-alert>
  </div>
</ComponentDemo>

## With Actions

Use the `actions` slot to add action buttons inside the alert.

<ComponentDemo title="With Actions">
  <hx-alert variant="warning" dismissible>
    Your session is about to expire.
    <div slot="actions">
      <hx-button variant="secondary" size="sm">
        Extend Session
      </hx-button>
    </div>
  </hx-alert>
</ComponentDemo>
