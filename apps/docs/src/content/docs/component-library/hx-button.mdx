## Loading

Buttons in loading state show a spinner and prevent interaction.

<ComponentDemo title="Loading">
  <div style="display: flex; gap: 0.75rem; flex-wrap: wrap; align-items: center;">
    <hx-button variant="primary" loading>
      Saving...
    </hx-button>
    <hx-button variant="secondary" loading>
      Loading
    </hx-button>
  </div>
</ComponentDemo>

## With Prefix and Suffix Slots

Use `prefix` and `suffix` slots to add icons or content before/after the label.

<ComponentDemo title="Prefix & Suffix">
  <div style="display: flex; gap: 0.75rem; flex-wrap: wrap; align-items: center;">
    <hx-button variant="primary">
      <span slot="prefix">+</span>
      Add Item
    </hx-button>
    <hx-button variant="secondary">
      Download
      <span slot="suffix">↓</span>
    </hx-button>
  </div>
</ComponentDemo>

## As Link

Set `href` to render the button as an anchor element.

<ComponentDemo title="Link Button">
  <hx-button variant="primary" href="https://example.com" target="_blank">
    Visit Site
  </hx-button>
</ComponentDemo>
